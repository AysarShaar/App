# 🎓 دليل تحويل منصة "دورات الوطني" إلى تطبيق أندرويد باستخدام Flutter

لقد قمنا بتحويل الهيكل البرمجي بالكامل لمشروع الويب الحالي إلى تطبيق أندرويد حقيقي (Native Android App) مكتوب بلغة **Flutter/Dart**. يتضمن المشروع تفعيل ترخيص الأكواد في الخلفية، تصفح المناهج والسنوات بشكل ذكي، المفضلة، والملاحظات الشخصية.

---

## 🏗️ 1. هيكل المجلدات والملفات المُنظم بالكامل (الذي تم إنشاؤه):

* **`pubspec.yaml`**: يحتوي على جميع المكتبات كالدخول لبيانات الهاتف لتأمين التراخيص (`device_info_plus`) وقاعدة البيانات في الخلفية ومكتبة قراءة الـ Markdown لتثبيت النصوص الأكاديمية والطبية.
* **`lib/main.dart`**: المدخل وتهيئة خدمات Firebase والتحقق اللحظي التلقائي.
* **`lib/models/question_model.dart`**: الكائنات المترجمة عن هيكل التصنيفات والسنوات والأسئلة.
* **`lib/services/local_storage_service.dart`**: خدمة الحفظ المحلي (الكاش) المتكامل للـ Streak والمفضلة وترخيصك.
* **`lib/services/firebase_service.dart`**: للاتصال السحابي بقواعد بيانات التراخيص والمزامنة.
* **`lib/views/`**:
  * `main_navigation_shell.dart`: شريط التنقل السفلي الذكي لجميع التبويبات.
  * `home_view.dart`: لوحة التصنيفات والدورات السنوية والاختبارات السريعة.
  * `quiz_view.dart`: منصة الأسئلة الممتعة والعداد والملاحظات وإظهار النتيجة.
  * `search_view.dart`: محرك البحث الذكي الفوري في الكلمات ومحتوى الأسئلة.
  * `favorites_view.dart`: استعراض وحفظ تفاصيل دراستك ومفاتيح المراجعة.

---

## 📱 2. كيف تصدر التطبيق كملف APK للأندرويد؟

لتصدير التطبيق الخاص بك كملف تثبيت أندرويد (**APK**)، اتبع الخطوات التالية بدقة:

### أ) متطلبات بيئة العمل الأساسية:
1. قم بتحميل وتثبيت **Flutter SDK** على حاسوبك من الموقع الرسمي.
2. قم بتحميل وتثبيت **Android Studio** لتهيئة بيئة أدوات الأندرويد (Android SDK, Build Tools).
3. يفضّل تثبيت محرر أكواد خفيف مثل **VS Code** أو العمل عبر Android Studio مباشرةً.

### ب) إنشاء المشروع وتثبيت الأكواد:
1. افتح مبدل الأوامر (Terminal) في جهازك وقم بإنشاء مشروع جديد بمطابقة معرّف تطبيقك (مثال: `com.national.quiz`):
   ```bash
   flutter create national_quiz_app --org com.national
   ```
2. قم بنسخ مجلد `flutter_project/lib` وملف `pubspec.yaml` اللذين قمنا بإنشائهما مسبقاً، واستبدل بهما تلك الموجودة في مشروعك الجديد.
3. قم بتثبيت مراجع ومكتبات الأكواد عبر تشغيل الأمر:
   ```bash
   flutter pub get
   ```

### ج) الأوامر السحرية لتصدير ملف الـ APK:

يحتوي الفلاتر على عدة خيارات للتصدير بحسب الحاجة:

#### 1️⃣ الطريقة السريعة (Debugging/التجربة):
إذا كنت تريد تصدير ملف سريع لإرساله للطلبة للتجربة السريعة على هواتفهم، استخدم الأمر التالي:
```bash
flutter build apk --debug
```
* **المميزات:** سريع البناء ويقبل التثبيت المباشر.
* **مكان الملف الناتج:** `build/app/outputs/flutter-apk/app-debug.apk`

#### 2️⃣ الطريقة الرسمية للنشر (Release) - مدمجة ومحسنة:
لإنشاء تطبيق نهائي بمساحة صغيرة جداً وسرعة عالية للأجهزة الذكية:
```bash
flutter build apk --release
```
* **المميزات:** يقوم بضغط حجم الأكواد وتشغيل محرك التشفير وتثبيت أداء الرسوم الفائقة.
* **مكان الملف الناتج:** `build/app/outputs/flutter-apk/app-release.apk`

#### 3️⃣ التصدير المتوافق مع جميع المعالجات (Split Per ABI):
بعض الهواتف القديمة تعمل بمعالجات 32-بت والجديدة 64-بت، لتصدير ملفات مستقلة ومصغرة لكل معالج:
```bash
flutter build apk --split-per-abi
```
* **سينتج هذا الأمر ثلاثة ملفات:** نسخة لمعالجات v7, v8 و x86 لتقليص المساحة لأقل حد ممكن.

## 🏆 ملامح ميزات الحساب والتشغيل السريع في الخلفية:
قمنا بمطابقة لمسات اللوجيك الويب، حيث عند تشغيل التطبيق في فلاتر:
1. يقرأ الـ **Splash/Main** مباشرة من الذاكرة المحلية المخزنة مسبقاً في دقيقة واحدة.
2. يدخل الطالب دون انتظار على الإطلاق لحل الأسئلة.
3. يعمل التطبيق في الخلفية (Background Execution) للاستعلام عن كود التفعيل وصلاحية الحساب، مما يمنع تعليق أو تعطيل دراسة الطالب بسبب اتصال الإنترنت ضعيف أو منقطع.
